# MonitoringWaterTemp

Dengan menggunakan DS18B20 ini kita dapat melakukan monitoring suhu air.



Alat / Bahan
           
1.Arduino Uno                                    
2.Sensor DS18B20 Waterproof          
3.Resistor 220 ohm                           
4.Board                                 
5.Jumper     
6.Arduino IDE                                     
7.Library OneWire                              
8.Library DallasTemperature
